# PDF Merger Web App
## A fast and efficient web application for merging multiple PDF files into a single document instantly

## About The Project
PDF Merger is a user-friendly web application that allows users to seamlessly combine multiple PDF documents into one. Built with a clean interface and powerful backend processing, it handles PDF merging quickly and reliably.

## Live Website: 

### Key Features
#### Instant Merging: Combine multiple PDF files in seconds

#### Drag & Drop Interface: Easy file selection with Bootstrap UI

#### Server-Side Processing: Efficient PDF handling using pdf-lib

#### No Quality Loss: Maintains original PDF quality

#### Secure Processing: Files processed securely on the server

#### Responsive Design: Works perfectly on desktop and mobile devices
